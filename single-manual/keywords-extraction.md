# 文本关键词提取

## 说明

## API URL
```
https://innovaapi.aminer.cn/tools/v1/extra/word/extract_kw
```

## HTTP Method

```
POST
```

## 输入参数

| 名称             | 说明                      |
| ---------------- | ------------------------- |
| text             | 中英文文本，比如"表示学习旨在将研究对象的语义信息表示为稠密低维实值向量,知识表示学习则面向知识库中的实体和关系进行表示学习" |
