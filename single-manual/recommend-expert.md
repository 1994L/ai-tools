# 专家推荐

##说明

## API URL
```
https://innovaapi.aminer.cn/tools/v1/predict/experts
```

## HTTP Method

```
POST
```

## 输入参数

| 名称             | 说明                      |
| ---------------- | ------------------------- |
| num              | 推荐专家人数，比如 20        |
| text             | 相关专家文本数据，比如 "本项目通过对聚电解质材料的溶液性质、流变特征和功能构效关系进行系统研究，发展了聚电解质基本理论和流变模型。" |
